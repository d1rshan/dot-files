## Structure

```text
.config/
├── fastfetch/      # fetch config
├── fish/           # shell
├── hypr/           # WM, lock, idle, paper
├── kitty/          # terminal
├── mako/           # notifications
├── nvim/           # editor
├── rofi/           # launcher + themes
├── waybar/         # status bar
└── starship.toml   # prompt
```
